# TournamentMaker

## Installation
```sh
clone https://github.com/silentx1981/tournamentMaker.git
cd tournamentMaker
composer install
```

Der ```web```-Path muss auf ```public_html``` gesetzt werden.

## Konfiguration
_config/host.local_
```json
{
    "basePath": "/",
    "images": {
        "logo": "images/logo.png"
    },
    "name": "TournamentMaker"
}
```
